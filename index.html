What It Does

FEATURES

01
Advanced Search

Filter by model, city, year, body style, transmission and price range with real-time results.

02
Multi-Image Gallery

Each listing supports multiple images with a slider gallery on the detail page.

03
Inquiry System

Registered users can send one inquiry per car. All inquiries visible in personal dashboard.

04
User Auth

Full register, login, logout and personal dashboard for managing inquiries.

05
Featured Cars

Mark cars as featured from admin panel. Featured cars appear prominently on homepage.

06
Admin Panel

Full Django admin with inline image management, bulk actions, and colored status badges.

07
Team Showcase

Manage team members from admin. Displayed on the About page with social links.

08
Contact Form

Visitors can send messages directly. All contact submissions tracked in admin.

Get Started

INSTALLATION

# Clone the repository
$git clone https://github.com/alisamadzadeh46/car.git && cd car
# Create virtual environment
$py -m venv .venv && .venv\Scripts\Activate
# Install dependencies
$pip install -r requirements.txt
# Run migrations & create superuser
$python manage.py migrate && python manage.py createsuperuser
# Start the server
$python manage.py runserver
